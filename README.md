# BMW Website 🚗

A modern BMW-inspired website built using **HTML, CSS, and JavaScript**.
This project showcases BMW models, videos, and a clean responsive design.

## 🌟 Features

* Responsive Navigation Bar
* Hero Section with Call-To-Action
* BMW Models Showcase
* Video Reels Section
* Smooth Animations
* Dark/Light Theme Toggle
* Back to Top Button
* Footer with Contact and Social Links

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
